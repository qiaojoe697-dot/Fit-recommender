# 设计决策记录

## 技术选型
| 决策 | 选择 | 理由 |
|------|------|------|
| 语言 | Python | 代码量少、迭代快，聚焦 Loop 设计学习 |
| 推荐算法 | 基于内容的属性相似度 | 无需训练数据，适合课程作业规模 |
| CLI 库 | argparse（标准库） | 零额外依赖 |
| Web UI | Streamlit | 快速出界面，适合第5轮扩展 |
| 数据格式 | JSON | 易于手工构造和维护 |

## 算法设计
- 采用加权评分而非纯余弦相似度，便于理解和调参
- 颜色和谐度使用预定义色系相似矩阵（非实际图像处理）
